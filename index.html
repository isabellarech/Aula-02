Parágrafo

No HTML, a tag "p" é utilizada para a criação de parágrafos.
Além disso, ela auxilia na organização do texto em blocos estruturados, reunindo frases, imagens ou outros componentes relacionados.
Ela insere automaticamente um espaçamento vertical (margem) antes e depois do conteúdo, fazendo com que a leitura seja de fácil entendimento.

Horizontal Ruler (hr)

A tag "hr" cria uma linha horizontal para separar temas ou seções diferentes no texto.
Portanto, é ideal para dar uma pausa visual e organizar melhor o conteúdo da página.

Tag de Abreviações e Acrônimos

A tag "abbr" é utilizada quando uma palavra é uma abreviação ou acrônimo, como
"HTML", "SP" ou "CSS".
Ela serve para deixar o texto mais direto, indicando que o termo representa apenas
uma versão abreviada de algo mais extenso.
Além de facilitar a organização da informação apresentada, facilita buscadores,
navegadores e até mesmo leitores de tela a compreenderem precisamente o seu significado.
